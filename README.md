# 🛍️ Myntra Clone JS

🌐 Live Demo: https://anjali-singhal-code.github.io/Myntra-clone-js/

A responsive and interactive Myntra inspired shopping website built using HTML, CSS, and JavaScript. This project includes separate product sections, cart functionality, and dynamic price calculation.

---

## 🚀 Features

- 🛒 Add to cart functionality
- 💰 Dynamic total price calculation
- 👕 Separate Men and Women product sections
- 📱 Responsive design
- 🎨 Modern shopping UI
- ⚡ Interactive user experience

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

## 📂 Project Structure

- `index.html`
- `products.html`
- `cart.html`
- `style.css`
- `products.css`
- `cart.css`
- `script.js`
- `products.js`
- `cart.js`

---

## 📌 Purpose of Project

This project was created to practice DOM manipulation, cart functionality, dynamic rendering, and responsive e-commerce UI development using JavaScript.
